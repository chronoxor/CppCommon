Detailed Description

template<typename T, class TMemoryManager, bool nothrow = false>
class CppCommon::Allocator< T, TMemoryManager, nothrow >

Memory allocator wrapper class.

Memory allocator wrapper implements standard allocator interface and wraps the memory manager provided as a template argument.

Not thread-safe.

Examples:
memory_arena.cpp, and memory_pool.cpp.
