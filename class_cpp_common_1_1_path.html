Detailed Description

Filesystem path.

Filesystem path wraps string directory, filename, symlink and other path types in a class and provide utility methods to check type of the path, status and permissions in a file system. Additionally path wrapper contains operators and methods for path manipulation (concatenation, canonization, absolute path).

Path is managed in UTF-8 encoding!

Not thread-safe.
