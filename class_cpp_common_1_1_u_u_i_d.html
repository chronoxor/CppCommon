Detailed Description

Universally unique identifier (UUID)

A universally unique identifier (UUID) is an identifier standard used in software construction. This implementation generates UUID with the system provided functionality (CoCreateGuid(), uuid_generate(), etc).

A UUID is simply a 128-bit value: "123e4567-e89b-12d3-a456-426655440000"

Not thread-safe.
