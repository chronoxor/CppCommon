Detailed Description

template<class TAuxMemoryManager = DefaultMemoryManager>
class CppCommon::PoolMemoryManager< TAuxMemoryManager >

Memory pool manager class.

Memory pool manager uses a pre-allocated memory buffer or several memory chunks in order to create an effective free-list data structure, that allows to allocate and free memory.

During the allocation memory pool manager will return a first-fit memory block in the free list!

If the allocated block is huge and does not fit into the memory pool chunk then it will be allocated directly from auxiliary memory manager.

Not thread-safe.
